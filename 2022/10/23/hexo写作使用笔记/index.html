使用hexo进行博客写作

1.创建文章或页面

layout默认为post，可以在_config.yml文件中修改

2.布局（Layout）

Hexo 有三种默认布局：post、page 和 draft，创建时会保存到不同的路径中

  • post —-> source/_posts
  • page —-> source
  • draft —–> source/_drafts

3. 文件名称

默认以标题为文件名。

4.草稿

使用hexo publish [layout] <title>发布草稿

5.模板

根据scaffods文件夹对应的文件建立文件

6.服务器

  • 启动服务器 hexo server,会在locakhost:4000下启动
  • 更改端口：hexo server -p 5000
  • 静态模式: hexo server -s
  • 自定义ip： hexo server -i 192.168.1.1

7.生成文件

  • hexo generate 生成静态文件
  • hexo generate --watch 监视文件变得并立即重新生成静态文件。
  • hexo gegerate --deploy 在生成完毕后自动部署，可简写为hexo g -d

8.部署

  • 一键部署： hexo deploy

9.参考文档

  • 官方教程：文档 | Hexo
